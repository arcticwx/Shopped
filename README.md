# Shopped
## Shopped is a shop management and tracking system.

Features:
    <ul>
        <li>
        Add sale
        </li>
        <li>
        Add cost        
        </li>
        <li>
        Keep track of sales and cost, to track profit
        </li>
        <li>
        Edit inventory - to keep track of it.
        </li>
    </ul>

<br>

Made with C++.
